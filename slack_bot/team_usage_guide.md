# 🎨 ZeroDeck 봇 사용법 (팀원용)

ZEROBASE 디자인 시스템 기반 PPT를 슬랙에서 바로 만들 수 있어요.

---

## 4가지 사용 방법

### 1. 🏃 가장 빠르게 — 슬래시 커맨드

```
/제로덱 자기소개서 작성법, 신입 취준생 대상, 8장
```

→ AI가 구조 잡고 디자인 입혀서 1~2분 안에 결과 회신.

### 2. 💬 대화형 — 봇 멘션

```
@ZeroDeck 면접 준비 자료 만들어줘
```

→ 봇이 응답하면 추가 정보 자연어로 주고받기.

### 3. 📋 정밀 입력 — 폼

```
/제로덱-폼
```

→ 모달 창이 뜸. 주제·대상·챕터·톤 등 정확히 입력.

### 4. 📁 직접 작성한 JSON — 파일 업로드

이미 JSON 스키마를 아는 분은 `.json` 파일을 채널에 업로드하기만 하면 됩니다. 봇이 자동 감지해서 빌드.

---

## 결과물 (3종 자동 생성)

매 요청마다 다음 3개를 받습니다:

- 🌐 **HTML 미리보기 링크** — 브라우저에서 바로 확인, 발표자 모드 지원
- 📑 **PPTX 파일** — Google Slides·PowerPoint에서 편집 가능
- 📄 **PDF 파일** — 인쇄·공유용 고해상도

---

## 입력 팁

좋은 결과를 얻으려면:

✅ **명확한 주제 + 대상**
> "자기소개서 작성법, 신입 대상"

✅ **슬라이드 수 지정**
> "...8장 정도로"

✅ **톤·스타일**
> "...컨설팅 톤으로" / "...친근하게"

✅ **참고 자료 언급** (Case 2, AI 분석)
> "이 합격 사례들 분석해서 만들어줘 — [PDF 첨부]"

❌ **너무 짧으면**
> "자소서" → AI가 임의 해석

---

## 자주 묻는 질문

**Q. 봇이 응답이 없어요**
→ 먼저 `/제로덱-도움말`로 봇이 깨어있는지 확인. 응답 없으면 `#zerodeck-issues` 채널에 문의.

**Q. 결과물이 마음에 안 들어요**
→ 다시 명령어로 요청하세요. AI는 매번 다르게 생성합니다. 또는 PPTX를 다운받아 직접 수정 가능.

**Q. 회사 내부 자료(합격 사례 PDF 등)를 분석하게 하고 싶어요**
→ 채널에 PDF 업로드 + 봇 멘션 → "이 자료 참고해서 자소서 가이드 만들어줘"

**Q. 디자인을 바꿀 수 있나요?**
→ 디자인 시스템(v1.4) 내에서만. 색상·폰트·레이아웃은 통일 유지.

**Q. 문제가 생기면?**
→ Slack 메시지 캡처해서 `#zerodeck-issues` 또는 직접 DM.
